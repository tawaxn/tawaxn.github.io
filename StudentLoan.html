StudentLoan MFU

Student Loan Fund System

Project Overview

StudentLoan is a website designed to simplify the student loan process. The system covers everything from loan applications and disbursements to user management. This project was developed to demonstrate skills in database management and backend logic implementation.

Responsibilities: Responsibilities: In this project, I was responsible for adding various functions, managing the system and API integration between the frontend and backend.

Features

  • Loan application registration
  • View loan application status
  • Loan provider management to simplify the lending process
  • Admin management system

Technology used

HTML5, CSS3, JavaScript, Vue.js, Node.js, MySQL
