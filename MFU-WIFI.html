MFU WIFI

Wi-Fi Login Website with ThaiD Facial Recognition

Project Overview

MFU WI-FI is a project that develops an identity verification system for accessing Wi-Fi at Mae Fah Luang University and the Mae Fah Luang University Medical Center, using facial recognition technology through the ThaID application to make Wi-Fi access more convenient for external users.

Responsibilities: In this project, I was responsible for managing the database system, integrating APIs, and making minor frontend modifications.

Features

  • Scan the face through ThaID to connect to wifi.
  • High Security Verify your identity through ThaID, which is reliable and meets government security standards.
  • Wi-Fi Usage Duration After successful facial scanning, users can access the Wi-Fi for 1 hour.

Technology used

HTML5, Tailwind CSS, JavaScript, Face API, React, MongoDB, Node.js
