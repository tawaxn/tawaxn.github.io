Musicaial Rent Shop Mobile

Musical Instrument Borrowing Application

Project Overview

Musical Rent Shop Mobile is an application developed to allow users to conveniently borrow products and enable administrators to efficiently manage them.

Responsibilities: In this project, I was responsible for designing the frontend according to the design, developing the backend, connecting APIs, integrating with the database, and managing the system.

Features

  • Registration and Login System
  • Product Borrowing System
  • Dashboard System
  • Product Borrowing History System
  • Admin Management System

Technology used

HTML5, CSS3, JavaScript, Flutter, Node.js, MySQL
